# My Personal Card

An interactive personal card with animations, particle effects and a QR code, built with Next.js, React and Tailwind CSS.  

## Live Demo
[my-personal-card-lac.vercel.app](https://my-personal-card-lac.vercel.app)

## Features
- Interactive flip card UI
- Particle background using tsParticles  
- Smooth animations with Framer Motion  
- QR code integration for quick sharing  
- Fully responsive and mobile-friendly  

## Tech Stack
- Next.js
- React 19  
- Tailwind CSS  
- Framer Motion 
- tsParticles  
- React Icons

A fun-project exploring animations, interactive UI and React libraries.
